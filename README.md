# Tech-Bridge-Honeypot
Project Name: Tech Bridge Educational Honeypot

Description:
A multi-stage educational web honeypot designed to simulate a realistic school portal environment. The honeypot captures attacker login attempts, records GeoIP data, and provides administrative monitoring dashboards.

Features:
- Role-based login (Admin, Super User, Student)
- Fake course system (AWS & Cybersecurity)
- GeoIP tracking of login attempts
- Admin dashboard monitoring
- Credential capture logging
- Fake student database
- Realistic school UI

Technologies Used:
- Python
- Flask
- SQLite
- GeoIP2
- HTML/CSS
- Windows 11 VM
